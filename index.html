How to Get Songs Playing
The game cannot stream from YouTube directly — browsers block it for copyright reasons. Here's how to get audio working:
Step 1 — Download the songs as MP3
Use yt-dlp (free, open source) or an online tool like cobalt.tools to download audio from YouTube as MP3. Rename each file to lowercase with underscores — no spaces.
Step 2 — Name them exactly like this
love_you_like_an_alcoholic.mp3
masochistic_tango.mp3
faouzia_tears_of_gold.mp3
faouzia_born_without_a_heart.mp3
faouzia_you_dont_even_know_me.mp3
the_hollow_season.mp3
faouzia_minefields.mp3
faouzia_anybody_out_there.mp3
you_are_the_missing_singer.mp3
Step 3 — Place them in your repo
/your-repo/
  index.html ← this file
  /audio/
    love_you_like_an_alcoholic.mp3
    faouzia_tears_of_gold.mp3
    ... etc
Step 4 — Open the file locally first to test
Open index.html directly in Chrome/Firefox (file://). If audio plays locally, it will play on GitHub Pages too. Check the browser console for 404 errors if it doesn't load.
For uploaded songs: drag any MP3/WAV/OGG into the Upload section. M4A may fail in some browsers — convert to MP3 first using ffmpeg or an online converter.

before you begin
The Willow is not a game about perfection.

It listens. It remembers. It adapts.

Each memory you enter is played through rhythm — follow what you see, not what you expect. There are no instructions on screen. The flowers bloom when it is time. Trust that.

You may sing if you wish. You may stay silent. Both have consequences.

Your performance shapes what the archive shows you. Clean execution may feel correct, but it will not always reveal the truth. Imperfection has a way of breaking through.

Some details will change the second time. Others will lie to you the third.

If you hesitate, it will notice. If you fail, it may keep you.

Nothing here is entirely real. But something here is remembering you.
